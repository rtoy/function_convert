Function: sinc (x) ¶

Return sin(x)/x. The function sinc(x) is defined for all real and complex values of x. At x = 0 the value is defined to be 1, corresponding to the limiting value of sin(x)/x.

When sinc is applied to a symbolic expression, the result is the quotient sin(x)/x. When $numer is true and the argument is a floating or complex floating number, sinc performs floatingâ€‘point evaluation.
